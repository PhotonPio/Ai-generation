# Music Tracks

Add 5-10 CC0 `.mp3` files in this folder.

Recommended naming pattern so theme matching works:

- `calm_space_01.mp3`
- `calm_history_01.mp3`
- `epic_cosmic_01.mp3`
- `epic_battle_01.mp3`
- `uplifting_science_01.mp3`

The renderer auto-selects a theme-matching track and mixes it at 15% volume.
If this folder has no `.mp3`, the pipeline falls back to generated sine-wave music.
